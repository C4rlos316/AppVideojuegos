# Contribuir

## Ramas
- `feat/<scope>-<resumen>`
- `fix/<scope>-<resumen>`
- `chore/<scope>-<resumen>`

## Commits (Conventional Commits)
- `feat(ui): agrega grid responsive de Home`
- `fix(services): corrige manejo de errores RAWG`

## PRs
- Use la plantilla.
- Requiere CI verde y revisión.

## Calidad
- `npm run lint` y `npm run check` obligatorios.
- Cobertura mínima para módulos críticos cuando existan pruebas.
